Privacy Policy

What we collect

We do not collect personal data, set cookies, or use third-party analytics trackers.

  • Video URLs you submit are processed in-memory and are not logged or stored on any server.
  • Sentiment corrections you make are saved in a local JSON file on the machine running the app. They are never transmitted to any external service.

Third-party services

  • YouTube / Google — comments are fetched via YouTube’s InnerTube API through yt-dlp. Your IP address is visible to YouTube during this request.
  • Hugging Face — the AI model is downloaded once from Hugging Face Hub and cached locally. No comment data is sent to Hugging Face.

Data retention

Analysis results exist only for the duration of the HTTP response. Once you close or navigate away from the results page, the data is gone. Only user-submitted sentiment corrections persist locally.
